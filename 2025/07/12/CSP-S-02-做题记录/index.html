具体的扩容方法为：枚举当前容量 SSS 中每一位的数值。

  • 如果当前数值为 111，扩容后当前这一位的值依然是 111。
  • 如果当前数值为 222，扩容后当前这一位的值会扩展为 222222。
  • 如果当前数值为 333，扩容后当前这一位的值会扩展为 333333333。

题意：有 NNN 个数，每个数设置为红或者蓝。

1．如果 AiA_iAi​ 左侧没有同色的数，Ci=0C_i=0Ci​=0。

20pts： 枚举每个数颜色取 111，和取 222 这两种情况的所有的组合，求每种组合下，分数的最大值。

1．状态定义

f[i][0]f[i][0]f[i][0]：前 iii 个数，如果第 iii 个数对于答案没有贡献，最大得分。
f[i][1]f[i][1]f[i][1]：前 iii 个数，如果第 iii 个数对于答案有贡献，最大得分。

要注意：并不是第 iii 个数可以对答案有贡献时（即：左侧有等值的数），产生贡献一定最优。

比如：1 2 3 3 3 2 11\ 2\ 3\ 3\ 3\ 2\ 11 2 3 3 3 2 1，对于第 666 个数，如果要对答案有贡献，那么 2 3 3 3 22\ 3\ 3\ 3\ 22 3 3 3 2 必须同色，此时得分 =0+0+0+3+0+1=4= 0 + 0 + 0 + 3 + 0 + 1 = 4=0+0+0+3+0+1=4 分。

而让第 666 个数不产生贡献，答案反而更大，即：染色方案为 1 2 3 3 3 2 11\ 2\ 3\ 3\ 3\ 2\ 11 2 3 3 3 2 1，此时得分 =0+0+0+3+2+0=5= 0 + 0 + 0 + 3 + 2 + 0 = 5=0+0+0+3+2+0=5 分。

2．状态转移

(1) 如果第 iii 个数对于得分无贡献。

  • 等价于第 iii 个数不存在：f[i][0]=max⁡(f[i−1][0],f[i−1][1]);f[i][0] = \max(f[i - 1][0], f[i - 1][1]);f[i][0]=max(f[i−1][0],f[i−1][1]);

(2) 如果第 iii 个数对于得分有贡献。

  • 此时需满足 a[i]a[i]a[i] 左侧有和其值相等的数，假设 a[i]a[i]a[i] 左侧和其值相等的最近的数的位置是 ppp。

  • 则有：ppp 和 iii 必然同色，且 p+1∼i−1p+1 \sim i-1p+1∼i−1，必然和 p/ip/ip/i 异色。

  • 则到 iii 为止的最大得分可以分为 3 部分讨论。

    • 第一部分：第 iii 个数的贡献，就是 a[i]a[i]a[i]。

    • 第二部分：p+2∼i−1p+2 \sim i-1p+2∼i−1 的贡献，这部分只能是每个数 ====== 其左侧的数，才有贡献，可以通过前缀和求区间和维护出来。

    • 第三部分：1∼p+11 \sim p+11∼p+1 的贡献 =max⁡(f[p+1][0],f[p+1][1]);= \max(f[p + 1][0], f[p + 1][1]);=max(f[p+1][0],f[p+1][1]);

为什么有贡献时，一定是从最近的等值的数转移过来最优？

设 iii 的左侧有多个和其值相等的点，比如有 ppp 和 p′p'p′，如果从 p′p'p′ 转移过来，导致从 p′+1∼p+1p'+1 \sim p+1p′+1∼p+1 这段必须只能是同色，而同色只是所有颜色排列的情况之一，讨论了更少的情况不可能产生更优解。

T4 P8819 [CSP-S 2022] 星战

题目描述

在这一轮的星际战争中，我方在宇宙中建立了 nnn 个据点，以 mmm 个单向虫洞连接。我们把终点为据点 uuu 的所有虫洞归为据点 uuu 的虫洞。

战火纷飞之中这些虫洞很难长久存在，敌人的打击随时可能到来。这些打击中的有效打击可以分为两类：

  1. 敌人会摧毁某个虫洞，这会使它连接的两个据点无法再通过这个虫洞直接到达，但这样的打击无法摧毁它连接的两个据点。
  2. 敌人会摧毁某个据点，由于虫洞的主要技术集中在出口处，这会导致该据点的所有还未被摧毁的虫洞被一同摧毁。而从这个据点出发的虫洞则不会摧毁。

注意：摧毁只会导致虫洞不可用，而不会消除它的存在。

为了抗击敌人并维护各部队和各据点之间的联系，我方发展出了两种特种部队负责修复虫洞：

  • A 型特种部队则可以将某个特定的虫洞修复。
  • B 型特种部队可以将某据点的所有损坏的虫洞修复。

考虑到敌人打击的特点，我方并未在据点上储备过多的战略物资。因此只要这个据点的某一条虫洞被修复，处于可用状态，那么这个据点也是可用的。

我方掌握了一种苛刻的空间特性，利用这一特性我方战舰可以沿着虫洞瞬移到敌方阵营，实现精确打击。

为了把握发动反攻的最佳时机，指挥部必须关注战场上的所有变化，为了寻找一个能够进行反攻的时刻。总指挥认为：

  • 如果从我方的任何据点出发，在选择了合适的路线的前提下，可以进行无限次的虫洞穿梭（可以多次经过同一据点或同一虫洞），那么这个据点就可以实现反击。
  • 为了使虫洞穿梭的过程连续，尽量减少战舰在据点切换虫洞时的质能损耗，当且仅当只有一个从该据点出发的虫洞可用时，这个据点可以实现连续穿梭。
  • 如果我方所有据点都可以实现反击，也都可以实现连续穿梭，那么这个时刻就是一个绝佳的反攻时刻。

总司令为你下达命令，要求你根据战场上实时反馈的信息，迅速告诉他当前的时刻是否能够进行一次反攻。

  • 这道题的解法核心就在于暴力地去搜顺子，每搜完一层以后就贪心地出散牌。
  • 无论怎样来说，搜顺子有利于我们快速而轻松地打出 555 张甚至以上的牌，其余的东西，就留到最后慢慢打出就好了。
