RMQеӯҰд№ з¬”и®°

еүҚиЁҖ:иҝҷдёӘз®—жі•ж— и®әжҳҜд»ҺйҖӮй…ҚжҖ§иҝҳжҳҜй•ҝеәҰжқҘиҜҙйғҪеҫҲжңүе®һеҠӣвҖҰрҹ’Ұ

е…ідәҺ RMQ

RMQ жҳҜиӢұж–Ү Range Maximum/Minimum Query зҡ„зј©еҶҷпјҢиЎЁзӨәеҢәй—ҙжңҖеӨ§пјҲжңҖе°ҸпјүеҖјгҖӮ

иҜҰз»ҶдҝЎжҒҜ

жұӮ $l-r$ еҢәй—ҙеҶ…зҡ„жңҖеӨ§/жңҖе°Ҹж•°.

еҢәй—ҙжһ„йҖ 

  • жң¬иҙЁжҳҜDP.и®ҫ $f[i][j]$ дёә $i\sim i+2^{j-1}$ зҡ„еҢәй—ҙжңҖеӨ§еҖј.зү№еҲ«ең°,$f[i][0]=a[i]$.(дёҖдёӘж•°зҡ„жңҖеӨ§еҖјжҳҜе®ғжң¬иә«).
  • зҠ¶жҖҒиҪ¬з§»ж–№зЁӢ: $f[i][j]=max(f[i][j-1],f[i+2^{j-1}][j-1])$.

еҢәй—ҙжҹҘиҜў

  • и®ҫиҰҒжҹҘиҜўзҡ„еҢәй—ҙдёә $[L,R]$ (еҢ…жӢ¬LR)
  • $$k = log2(R-L+1)$$
  • $$res[L][R] = max(f[L][K],f[R-2^k+1][k])$$
  • жіЁж„Ҹ:дёәдҝқиҜҒзІҫеәҰ,иҜ·иҮӘе·ұжҺЁеҜј $2^k$ ,иҖҢдёҚжҳҜзӣҙжҺҘдҪҝз”Ё $log$ еҮҪж•°.

ж—¶й—ҙеӨҚжқӮеәҰ: $O(nlog_2n)$

A. и¶…зә§и®°еҝҶеҠӣ

йўҳзӣ®жҸҸиҝ°

е°ҸAеҗҢеӯҰжӢҘжңүж— дёҺдјҰжҜ”зҡ„и¶…зә§и®°еҝҶеҠӣпјҢд»–еҸҜд»ҘдёҖж¬ЎжҖ§и®°дҪҸеҫҲеӨҡж•°еӯ—гҖӮ

дёәдәҶиҖғйӘҢдёҖдёӢе°ҸAеҗҢеӯҰзҡ„и®°еҝҶеҠӣпјҢзҺӢиҖҒеёҲдёҖж¬ЎжҖ§з»ҷе°ҸAеұ•зӨәдәҶ $N$ дёӘж•ҙж•°гҖӮз„¶еҗҺй—®дәҶд»– $M$ дёӘй—®йўҳпјҢжҜҸдёӘй—®йўҳз»ҷе®ҡдёҖдёӘеҢәй—ҙпјҢиҰҒжұӮе°ҸAеҗҢеӯҰиҜҙеҮәиҝҷдёӘеҢәй—ҙдёӯзҡ„жңҖеӨ§ж•°жҳҜеӨҡе°‘пјҹ

дёәж–№дҫҝиҖҒеёҲжЈҖйӘҢе°ҸAеҗҢеӯҰзҡ„зӯ”жЎҲжҳҜеҗҰжӯЈзЎ®пјҢиҜ·дҪ е…Ҳзј–зЁӢжұӮеҮәжӯЈзЎ®зҡ„зӯ”жЎҲгҖӮ

иҫ“е…Ҙ

з¬¬дёҖиЎҢдёӨдёӘж•ҙж•° $N,M$ иЎЁзӨәж•°еӯ—зҡ„дёӘж•°е’ҢиҰҒиҜўй—®зҡ„ж¬Ўж•°пјӣ

жҺҘдёӢжқҘдёҖиЎҢдёә $N$ дёӘж•°пјӣ

жҺҘдёӢжқҘ $M$ иЎҢпјҢжҜҸиЎҢйғҪжңүдёӨдёӘж•ҙж•° $X,Y$ иЎЁзӨәиҜўй—®зҡ„еҢәй—ҙгҖӮ

ж•°жҚ®иҢғеӣҙпјҡ

$1вүӨNвүӨ10^5,1вүӨMвүӨ10^6,1вүӨXвүӨYвүӨN$гҖӮж•°еӯ—дёҚи¶…иҝҮ C/C++ зҡ„ int иҢғеӣҙгҖӮ

и§Јжі•:

  • жқҝеӯҗ,жІЎд»Җд№ҲеҸҜиҜҙзҡ„.жіЁж„Ҹ:иҰҒејҖscanf!

B. иҚЈиҖҖд№ӢжҲҳ

йўҳзӣ®жҸҸиҝ°

е°Ҹ $A$ жҳҜдёҖеҗҚжёёжҲҸзҺ©е®¶пјҢд»–жӯЈеңЁзҺ©дёҖж¬ҫеҸ«еҒҡвҖңиҚЈиҖҖд№ӢжҲҳвҖқзҡ„жёёжҲҸгҖӮеңЁиҝҷдёӘжёёжҲҸдёӯпјҢд»–йңҖиҰҒйҖҡиҝҮе®ҢжҲҗд»»еҠЎжқҘжҸҗеҚҮиҮӘе·ұзҡ„зӯүзә§гҖӮ

жёёжҲҸең°еӣҫдёҠжңү $N$ дёӘжҺ’жҲҗдёҖжҺ’зҡ„иЈ…еӨҮпјҢжҜҸдёӘиЈ…еӨҮйғҪж ҮжіЁеҘҪдәҶз»ҸйӘҢеҖје’ҢеҚұйҷ©еҖјпјҢз¬¬ $i$ дёӘиЈ…еӨҮзҡ„з»ҸйӘҢеҖјдёә $V_i$ еҚұйҷ©еҖјдёә $D_i$гҖӮ

е°Ҹ $A$ жҺҘеҲ°дәҶдёҖдёӘд»»еҠЎпјҢд»–йңҖиҰҒеңЁиҝҷ $N$ дёӘиЈ…еӨҮдёӯпјҢйҖүеҸ–иҝһз»ӯзҡ„иӢҘе№ІдёӘиЈ…еӨҮпјҢе№¶дҪҝеҫ—иҝҷдәӣиЈ…еӨҮзҡ„з»ҸйӘҢеҖјжҖ»е’ҢдёҚе°ҸдәҺ $K$пјҲ$\ge K$пјүгҖӮеҗҢж—¶иҰҒдҪҝеҫ—иҝҷдәӣиЈ…еӨҮзҡ„жңҖеӨ§зҡ„еҚұйҷ©еҖје°ҪеҸҜиғҪзҡ„е°ҸгҖӮ

иҜ·зј–зЁӢи®Ўз®—еҮәпјҢж»Ўи¶ійўҳж„Ҹзҡ„ж–№жЎҲдёӯпјҢжңҖеӨ§зҡ„еҚұйҷ©еҖјжңҖе°ҸжҳҜеӨҡе°‘пјҹ

иҫ“е…Ҙ

з¬¬ $1$ иЎҢиҜ»е…Ҙ $2$ дёӘж•ҙж•° $N,K$гҖӮ

жҺҘдёӢжқҘ $N$ иЎҢпјҢжҜҸиЎҢиҜ»е…Ҙ $2$ дёӘж•ҙж•° $V_i,D_i$ пјҢеҲҶеҲ«иЎЁзӨәжҜҸдёӘиЈ…еӨҮзҡ„з»ҸйӘҢеҖје’ҢеҚұйҷ©еҖјгҖӮ

ж•°жҚ®иҢғеӣҙ

еҜ№дәҺ $100%$ зҡ„ж•°жҚ®пјҢж»Ўи¶і $1 \le N \le 10^5$пјҢ$1 \le V_i,D_i \le 10^9$пјҢ$1 \le K \le 10^{18}$гҖӮ

  • иҰҒйҖүеҮәдёҖдёӘеҢәй—ҙ,дҪҝеҫ—д»–д»¬еңЁ $V_i\ge K$ зҡ„жғ…еҶөдёӢ $D_i$ зҡ„жҖ»е’ҢжңҖе°Ҹ.иғҪеӨҹиҪ¬еҢ–дёә: йҖүжӢ© $V_l+V_{i+1}+вҖҰ+V_r \ge K$ дё” $D_l+D_{i+1}+вҖҰ+D_r $ жңҖе°Ҹ.еҫҲе®№жҳ“жғіеҲ°з”ЁеҸҢжҢҮй’ҲжқҘи§ЈеҶі.жіЁж„Ҹ:дёҖе®ҡиҰҒи®©е·Із»ҸиҝҮеҺ»зҡ„ж•°еҮәйҳҹ!!!

C.ејӮжҲ–

йўҳзӣ®еҶ…е®№

жңүдёҖдёӘй•ҝеәҰдёә $N$ зҡ„ж•°еҲ— $A_1,A_2,\dots,A_n$гҖӮ

иҜ·й—®иҜҘж•°еҲ—дёӯд»»ж„ҸеҸ–дёҖдёӘеҢәй—ҙ $[L,R]$ дёӯпјҢжҳҜеҗҰеӯҳеңЁ $2$ дёӘж•°пјҢдҪҝеҫ—иҝҷдёӨдёӘж•°ејӮжҲ–зҡ„з»“жһңдёә $T$гҖӮ

иҜ·жіЁж„ҸпјҢжң¬йўҳдјҡеҸ‘иө· $M$ ж¬ЎиҜўй—®пјҢеҜ№дәҺжҜҸж¬ЎиҜўй—®зҡ„еҢәй—ҙпјҢеҰӮжһңиғҪжүҫеҲ°з¬ҰеҗҲйўҳж„Ҹзҡ„ж•°еҜ№пјҢиҜ·иҫ“еҮә yesпјҢеҗҰеҲҷиҜ·иҫ“еҮә noгҖӮ

иҫ“е…Ҙ

з¬¬дёҖиЎҢеҢ…еҗ«дёүдёӘж•ҙж•° $n, m, T$ гҖӮ

з¬¬дәҢиЎҢеҢ…еҗ« $n$ дёӘж•ҙж•°пјҢж•°еӯ—д№Ӣй—ҙз”Ёз©әж јйҡ”ејҖгҖӮ

жҺҘдёӢжқҘзҡ„ $M$ иЎҢпјҢжҜҸиЎҢжңүдёҖдёӘиҜўй—®еҢәй—ҙпјҢжҜҸдёӘиҜўй—®еҢәй—ҙеҢ…еҗ« $2$ дёӘж•ҙж•° $L,R$гҖӮ

ж•°жҚ®и§„жЁЎ

еҜ№дәҺ $20 %$ ж•°жҚ®, $1 \leq n, m \leq 100$;

еҜ№дәҺеҸҰеӨ– $40 %$ ж•°жҚ®, $1 \leq n, m \leq 1000$;

еҜ№дәҺ $100 %$ зҡ„ж•°жҚ®, $1 \leq n, m \leq 10^5,0 \leq T < 2^{20},1\le L \le R \le n,0 \le a_i \le 2^{20}$

жҖқи·Ҝ

  • йҰ–е…Ҳ,жҲ‘д»¬жҡҙеҠӣиӮҜе®ҡиҝҮдёҚдәҶ.
  • йўҳзӣ®иҰҒжұӮз»ҷе®ҡеҢәй—ҙеҶ…жҳҜеҗҰжңүдёҖеҜ№ж•°.$a,b$ ж»Ўи¶і $a^b=T$
  • з®ҖеҚ•жҺЁеҲ°еҸҜеҫ—: $a^T=b$ жҳҜз”ұдёҠйқўејҸеӯҗиҪ¬еҢ–иҖҢжқҘзҡ„.
  • жҲ‘д»¬йҡҸдҫҝеҲ—дёҖз»„ж•°жҚ®,е№¶еӯҳеӮЁеҲ° $a$ ж•°з»„йҮҢйқў: $a[]={2,1,3,4,2,3,2,3}$
  • з„¶еҗҺжІЎдёӘж•°ејӮжҲ– $T$ еҫ—: $a_2[]={3,0,2,5,3,2,3,2}$
  • з„¶еҗҺеңЁжҜҸдёӘж•°зҡ„еүҚйқўжҹҘжүҫејӮжҲ– $T$ еҗҺзҡ„ж•°,еҰӮжһңжүҫдёҚеҲ°ж Үи®°дёә $0$ :$a_3[]={0,0,1,0,3,5,6,7}$
  • еҰӮжһңиҰҒжүҫзҡ„еҢәй—ҙеҶ…еҜ№еә”зҡ„жүҖжңү $a_3[i]$ дёӯжңүеӨ§дәҺ $L$ иҖҢдё”е°ҸдәҺ $R$ зҡ„ж•°,йӮЈе°ұжҲҗеҠҹдәҶ.иҫ“еҮә $yes.$ еҗҰеҲҷе°ұжҳҜ $no$ .
  • иҝҷдёӘжүҫз¬ҰеҗҲи§„е®ҡзҡ„ж•°зҡ„иҝҮзЁӢеҸҜд»ҘдҪҝз”ЁRMQ.и®°еҪ•жҜҸдёӘеҢәй—ҙеҶ…зҡ„жңҖе°Ҹж•°з„¶еҗҺж–№дҫҝд№ӢеҗҺжҹҘжүҫ,е№¶дё”е…·жңүжңҖдјҳжҖ§.(дёҚдҝЎеҸҜд»ҘиҜ•иҜ•дёҚеҗҢзҡ„ж•°жҚ®)
